Detailed Description

A parity math component is a part of Motr core and serving several purposes:

  • Provide algorithms for calculation of SNS parity units (checksums) for given data units;
  • Provide algorithms for quick update of parity units in case of minor data changes;
  • Provide algorithms for SNS repair (recovery) in case of failure.
