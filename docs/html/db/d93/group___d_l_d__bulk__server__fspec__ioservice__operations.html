Detailed Description

See also
DLD of Bulk Server
Request handler

I/O Service initialization and operations controlled by request handler.

I/O Service defines service type operation vector -

  • I/O Service type operation m0_ioservice_alloc_and_init()
    Request handler uses this service type operation to allocate and and initiate service instance.

I/O Service defines service operation vector -

  • I/O Service operation m0_ioservice_start()
    Initiate buffer_pool and register I/O FOP with service
  • I/O Service operation m0_ioservice_stop()
    Free buffer_pool and unregister I/O FOP with service
  • I/O Service operation m0_ioservice_fini)
    Free I/O Service instance.

State transition diagram for I/O Service will be available at Request handler
