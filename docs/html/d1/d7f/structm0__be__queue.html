char* bq_qitems

Pre-allocated array of be_queue_item.

  • initially all items are in m0_be_queue::bq_q_unused;
  • when m0_be_queue_put() is called an item is moved to m0_be_queue::bq_q;
  • after m0_be_get() is done the item is moved back;
  • array of pointers to be_queue_item couldn't be used because the structure has a flexible array member at the end.
