bool spp_extra_ref

To add an extra reference on this record. When the record is consumed or it is stored in other persistent storage, pass the FDMI record ID back to this plugin (in stdin). At that time, plugin will release this FDMI record by calling the plugin dock API .fpo_release_fdmi_rec() .
