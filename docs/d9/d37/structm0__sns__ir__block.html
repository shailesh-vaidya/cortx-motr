Detailed Description

Every member of a parity-group is called as a block. During incremental recovery m0_sns_ir_block holds that information associated with a block which is relevant for recovery. This involves block-index within a parity-group, a pointer to information held by the block, a pointer to recovery-matrix associated with the block and so on.
