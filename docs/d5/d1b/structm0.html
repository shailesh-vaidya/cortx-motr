Detailed Description

m0 instance.

All "global" variables are accessible from this struct.

Each module belongs to exactly one instance.

m0 instance is allocated and initialised by external user. There are three use cases:

  • test creating one or more m0 instances (only one currently, but multiple instances could be very convenient for testing);
  • motr/setup creating a single m0 instance for m0d;
  • m0t1fs creating a single m0 instance for kernel.
